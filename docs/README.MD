# [安装npm工具](https://blog.csdn.net/zhangwenwu2/article/details/52778521)
```
# 支持工具, 如果npm工具没有，则可以安装node.js会自动拥有npm
npm i docsify docsify--cli -g --registry=https://registry.npm.taobao.org
#启动
docsify serve docs
docsify start docs
#访问：浏览器输入 http://http://localhost:3000
```
